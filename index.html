How it works

Bullseye runs silently while you play. It detects every ability cast and records whether it landed on an enemy champion — not minions, not walls.

  • Damage skillshots → accuracy % (landed / cast)
  • Utility abilities like W → usage count only
  • Healing abilities → total healed shown
  • Toggle on/off with Alt+B anytime
  • 100% transparent — never blocks gameplay
